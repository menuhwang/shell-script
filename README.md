# shell-script
각종 스크립트를 작성해둔 repo입니다.

# Usage
1. 리포지토리 클론
2. shell-script 폴더로 이동
3. 스크립트 실행
## 1. 리포지토리 클론
```sh
git clone https://github.com/menuhwang/shell-script.git
```

## 2. 폴더 이동
```sh
cd shell-script
```

## 3. 스크립트 실행

### Docker 최신 버전 설치
```sh
sh docker-install.sh
```

### Nginx 최신 버전 설치
```sh
sh nginx-install.sh
```
